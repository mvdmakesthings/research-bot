# Research Bot

A research bot using locally hosted LLMs, using Python and LangChain.

- Start by running `ollama run mistral`

---

## Getting Started
- Install [Ollama](https://ollama.ai/) and begin running your desired model:

    ```ollama run mistral```
- Run Main.py
